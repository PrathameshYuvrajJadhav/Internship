JavaScript Practical Test

Q1. Student Information Card

Create a webpage with two textboxes for Student Name and Course and one Submit button. When the button is clicked, display the entered information inside a Bootstrap card using JavaScript DOM manipulation.

Q2. Even and Odd Number Checker

Create a webpage with one textbox to enter a number and one button named Check. When the button is clicked, display whether the entered number is Even or Odd using JavaScript.

Q3. Dynamic List using Array

Create an array containing the names of five fruits. Using JavaScript and a loop, display all the fruit names inside an unordered list (<ul>) on the webpage.

Q4. Background Color Changer

Create a webpage containing three buttons: • Red • Green • Blue When the user clicks a button, change the background color of the webpage to the selected color using JavaScript and Event Listeners.

Q5. Store and Display User Name using Local Storage

Create a webpage with one textbox for entering a user's name and one Save button. Requirements: • Save the entered name in Local Storage. • Display the saved name below the textbox. • After refreshing the page, the saved name should still be displayed automatically.
